# Task 2: House Price Prediction (Regression)
Part of my Data Science internship at SoftGrowTech.

## 📊 Project Overview
This project focuses on building a predictive model to estimate house prices. It demonstrates the application of Linear Regression and feature relationship analysis.

## 🛠️ Tech Stack
- **Python** (Pandas, NumPy, Scikit-Learn)
- **Environment:** Google Colab

## 💡 Key Results
- **Algorithm:** Linear Regression.
- **Key Feature:** Identified 'Median Income' as the strongest predictor of house value.
- **Visualization:** Generated scatter plots with regression lines to show the correlation between income and price.
